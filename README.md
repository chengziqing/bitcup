特币宝 - 快乐世界杯
======

让2014的巴西世界杯更加欢乐！投注简单快捷、信息公开透明、中奖金额源地址返回！

今年世界杯期间,闲着无聊,就做了这个项目.项目很简单,使用比特币投注,然后根据比赛结果,猜对的人可以获得比特币奖金.主要思想是利用比特币的匿名特性,奖金自动返还到投资的比特币地址.界面采用bootstrap,紫色系,紫色有一种神秘特性.框架采用目前最有名的 revel.没有采用数据库.
